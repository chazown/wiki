# ⌨️ IDE & Productivity Shortcuts

## 1. 🟦 Visual Studio Code (Windows/Linux)
코딩 속도를 2배로 올려주는 필수 단축키 모음입니다.

### 🔹 필수 편집 (Editing)
| 기능 | 단축키 | 설명 |
| :--- | :--- | :--- |
| **명령 팔레트** | `Ctrl` + `Shift` + `P` | 모든 기능 검색 및 실행 (만능 키) |
| **파일 찾기** | `Ctrl` + `P` | 파일명으로 프로젝트 내 파일 즉시 이동 |
| **줄 복사** | `Alt` + `Shift` + `↓/↑` | 현재 줄을 위/아래로 그대로 복제 |
| **줄 이동** | `Alt` + `↓/↑` | 현재 줄을 위/아래로 이동 |
| **다중 커서** | `Alt` + `Click` | 여러 곳을 동시에 클릭하여 한 번에 편집 |
| **단어 선택** | `Ctrl` + `D` | 같은 단어를 하나씩 추가 선택 (반복 누름) |
| **주석 토글** | `Ctrl` + `/` | 선택 영역 주석 처리/해제 |
| **코드 정렬** | `Ctrl` + `K`, `F` | 흐트러진 들여쓰기 자동 정렬 |
| **사이드바 토글** | `Ctrl` + `B` | 왼쪽 파일 탐색기 숨김/표시 |

### 🔹 화면 및 터미널 (View)
| 기능 | 단축키 | 설명 |
| :--- | :--- | :--- |
| **터미널 열기/닫기** | `Ctrl` + ` (백틱)` | 하단 터미널 패널 토글 |
| **새 터미널** | `Ctrl` + `Shift` + ` (백틱)` | 터미널 탭 추가 |
| **터미널 분할** | `Ctrl` + `\` | 터미널 화면 좌우 분할 |
| **화면 분할** | `Ctrl` + `\` | 에디터 화면을 좌우로 분할 |
| **Markdown 미리보기** | `Ctrl` + `Shift` + `V` | 마크다운 파일 미리보기 탭 열기 |

## 2. 🌐 Chrome / Edge Browser
구글링과 자료 조사를 효율적으로 하기 위한 브라우저 단축키

### 🔹 탭 관리
* **`Ctrl` + `Shift` + `T`**: 방금 실수로 닫은 탭 다시 열기 (가장 중요!)
* **`Ctrl` + `W`**: 현재 탭 닫기
* **`Ctrl` + `Tab`**: 다음 탭으로 이동
* **`Ctrl` + `Shift` + `Tab`**: 이전 탭으로 이동
* **`Ctrl` + `1` ~ `8`**: 숫자 순서에 해당하는 탭으로 이동
* **`Ctrl` + `9`**: 무조건 마지막 탭으로 이동

### 🔹 주소창 및 기능
* **`Ctrl` + `L`**: 주소창으로 포커스 이동 (URL 복사 시 유용)
* **`Ctrl` + `F`**: 페이지 내 검색
* **`Ctrl` + `Shift` + `Delete`**: 인터넷 사용 기록(쿠키/캐시) 삭제 메뉴 진입
* **`F12`**: 개발자 도구 열기 (네트워크/콘솔 확인)
* **`Ctrl` + `Shift` + `N`**: 시크릿 모드(기록 안 남음) 새 창 열기

## 3. 🪟 Windows System
윈도우 환경에서 작업 효율을 높이는 시스템 단축키

### 🔹 클립보드 & 캡처
* **`Win` + `V`**: 클립보드 히스토리 (복사했던 내역 골라 붙여넣기)
* **`Win` + `Shift` + `S`**: 화면 캡처 (원하는 영역 지정)
* **`Win` + `.` (마침표)**: 이모지(😊) 입력창 열기

### 🔹 창 관리
* **`Win` + `L`**: 화면 잠금 (자리를 비울 때 필수)
* **`Win` + `D`**: 바탕화면 바로가기 (모든 창 최소화)
* **`Win` + `E`**: 파일 탐색기 열기
* **`Win` + `← / →`**: 현재 창을 화면 왼쪽/오른쪽으로 딱 붙이기 (분할 화면)
